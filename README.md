### Quickstart

### Prerequisites
[Node.js](https://nodejs.org) and [Angular CLI](https://github.com/angular/angular-cli).

### Installing
Install Node.js from [https://nodejs.org](https://nodejs.org).

Install Angular CLI `npm install -g @angular/cli`
